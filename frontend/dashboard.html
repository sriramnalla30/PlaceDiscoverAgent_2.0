SMS gateway

Outbound SMS uses the default agent phone on file (currently set in the backend settings). Shop contact numbers still come from SerpStack results.

To send from your own phone number (developer config):

  1. Install/configure the SMSMobileAPI app (or chosen provider) on your phone and obtain the API key.
  2. Share the key and your sender/receive number with the developer; they will set in .env:
    messaging_provider=smsmobileapi
    smsmobileapi_key=<provided_api_key>
    default_target_number=+<your_phone_with_country_code>
  3. Developer restarts the backend so the new gateway and number take effect.

If nothing is configured, the app uses the agent‚Äôs default gateway. Shop contact numbers still come from SerpStack results.
